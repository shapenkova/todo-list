# **SPA-приложение "Список заданий"**

## Как запустить?

1. Склонировать репозиторий к себе на ПК
2. npm install
3. npm run dev
4. Готово!

## Стэк технологий:
- Vue 3
- Vite
- Composition API
- Prettier
